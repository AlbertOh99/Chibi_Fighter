Game Name - Chibi Fighter

Albert Oh
Cameron Ahn
Matt Abbott-Rigsbee
Daniel Mares

The goal of the game is to defeat all of the enemies in the game in a beat em up style combat system

W - move up
A - move left
S - move down
D - move right

Space - attack (press again while in the middle of attack to do more attacks)

Cheat Codes - 
Ctrl + R = room restart

Ctrl + H = Full heal

Ctrl + P = Go to title screen

Ctrl + L = Destory all enemies
